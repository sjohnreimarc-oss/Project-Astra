# Project Astra

Project Astra is a Python-based orbital mechanics and mission planning toolkit inspired by realistic spaceflight simulation and aerospace engineering.

The goal of this project is to bridge physics, orbital mechanics, and programming into a single educational and practical application. It is designed primarily for enthusiasts of realistic spaceflight (such as RSS players in Kerbal Space Program), students interested in astrodynamics, and anyone curious about how spacecraft are planned and flown.

## Philosophy

Project Astra follows a simple philosophy:

- Small versions are good versions.
- One major physics feature per version.
- Every version must compile and work before moving on.
- Learn the mathematics before implementing it.
- Resist feature creep.
- Programming is the tool. Physics is the passion.

---

## Why This Project Exists

I wanted to build something that combines two things I enjoy most:

- Physics and orbital mechanics.
- Programming and software engineering.
- Inspired by being sick of writing solutions.
  
Instead of treating them as separate interests, Project Astra is my attempt at combining both into a practical and educational project.
